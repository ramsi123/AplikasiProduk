# Aplikasi Produk

Aplikasi Flutter sederhana untuk login. Menampilkan semua list produk, filter list produk, dan detail produk.

## Cara Menjalankan

1. Clone Repository  
git clone

2. Install Dependency  
flutter pub get

3. Menjalankan Aplikasi  
flutter run

## Plugin yang Digunakan

- http
- shared_preferences
